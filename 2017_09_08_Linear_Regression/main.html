Linear Regression with One Variable

This example demonstrates that - given a set of data of students' midterm and final exam's scores. By means of Gradient Descent algorithm, get the linear regression result to predict the students' final exam's scores using their midterm exam's scores.

In this simple application, you can append data on the graph by directly clicking on the chart or use the New Record button.

To generate the prediction result, you can click the Train Modal button.

If you want to reset and clear out all of the data, just click the Flush Data button.

If you want to see how gradient descent generate the result of regression line, try the Animate button.
